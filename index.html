🔗 Connect to Firebase
This tracker uses Google Firebase as a live database so everyone on your team sees the same data in real time. Setup takes about 5 minutes and is completely free.
1
Create a free Firebase project
Go to console.firebase.google.com and click Add project. Give it any name (e.g. "Sreenidhi Tracker"). Disable Google Analytics if asked — it's not needed.
2
Enable Firestore Database
In your project, go to Build → Firestore Database → click Create database. Choose Start in test mode (allows read/write for 30 days — you can secure it later). Select any region and click Enable.
3
Register a Web App & copy your config
In Project Settings (⚙ gear icon) → Your apps → click the </> Web icon. Give it a nickname, click Register app. You'll see a config object like this — copy the values into the fields below:
const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "your-project.firebaseapp.com", projectId: "your-project-id", storageBucket: "your-project.appspot.com", appId: "1:123456:web:abcdef" };
4
Paste your config below and connect
Fill in the fields below from the config above, then click Connect. Share this same HTML file with your team — they paste the same config and everyone works on the same live database.
